## ✨ FEATURES:
- **Dynamic Variable Fields**: Added support for variable fields in launchers. 
  - New prefix notation: Only shows if the variable field is not empty.
  - Added dedicated design and UI for variable entry.
- **Enhanced Web Launcher**:
  - Added dynamic URL detection.
  - New config option to specify a custom browser for `web_search` and bookmarks.
- **Weather Widget Overhaul**:
  - Added date and time display.
  - Added night-mode functionality.
  - Support for custom weather icon themes (automated via `gresources`).
- **Configuration Additions**:
  - `remember_query`: Option to remember the last searched query.
  - `lr_nav`: Added toggle for Left/Right arrow navigation in the search bar.
  - `n_clicks`: Configurable clicks required to execute a row.

## 🛠 Bug Fixes
- **Command Launching**: Fixed an issue where command arguments (especially those with nested quotes like Bottles/Flatpak) were not being split correctly.
- **Process Management**: 
  - Fixed incorrect subprocess forking and detachment issues.
  - Improved lockfile handling; now uses dynamic directories (`tmp`, `cache`, or `tmpdir`).
  - Fixed lockfile clearing when no PID exists.
- **UI/UX Stability**:
  - Resolved `BorrowMutError` panic during rapid typing.
  - Fixed focus loss when navigating from variable fields.
  - Fixed search bar "select all" behavior on focus grab.
  - Fixed case-sensitivity in search results.

## ⚡ Performance & Internal Changes
- **Binary Caching**: Switched from JSON to binary cache for improved performance.
- **Rust 2024 Edition**: Updated dependencies and migrated to the Rust 2024 edition.
- **Nixpkgs**: Updated to `nixpkgs-unstable` and renamed `wrapGAppsHook` to `wrapGAppsHook3`.
- **Startup Time**: Improved overall application initialization speed.

## 📝 Documentation
- **New Sections**: Added documentation for Aliases, Variable Fields, and New Keybind formats.
- **Assets**: Added `var-fields.svg` to visually explain variable field usage.
- **Fixes**: Cleaned up navigation typos and updated 404 links.
